# blch-evoting

A Hyperledger application for an evoting system, developed for the Blockchain module of FHNW.

## Quickstart

**Install and start the network**
```
npm install
npm run network-install
npm run network-start
npm run rest-start
```

**Start the application**
```
cd angular-app
npm start
```
